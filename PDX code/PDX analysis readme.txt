First place all of the following files in a single folder:

PDX-CT curve metrics.xlsx
Tumor types of models.csv
Well covered models.csv
Animal to animal consistency probability matrix.csv
BRAF mutations.csv
PDX analysis code.nb


The code can then be executed by loading the notebook 'PDX analysis code.nb' in Wolfram Mathematica v11.0, and selecting 'Evaluate Notebook' from the 'Evaluation' menu.



Source data files here produced from a subset of the supplementary data of:
Gao, H. et al. High-throughput screening using patient-derived tumor xenografts to predict clinical trial drug response. Nature Medicine volume 21, pages 1318-1325, doi:10.1038/nm.3954 (2015)
