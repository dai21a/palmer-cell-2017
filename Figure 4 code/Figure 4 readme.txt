First, place both of the following files in a single folder:

Table of survival distributions.xlsx
Figure 4, clinical trials analysis code.nb

The code can then be executed by loading the notebook 'Figure 4, clinical trials analysis code.nb' in Wolfram Mathematica v11.0, and selecting 'Evaluate Notebook' from the 'Evaluation' menu.



Note: in final copyediting of this article the sequence of Supplemental Figures S1 and S2 were swapped. Therefore where this code generates figures named 'Supplementary Figure S1' they appear in the article as Supplemental Figure S2.
